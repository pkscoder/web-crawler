# web-crawler

Getting Started
====================
- git clone https://github.com/priyamkrshaw/web-crawler.git
- Run `npm install`

Start Crawling
====================
- Run `node crawler.js <URL> <FILENAME>` or `node asyncCrawler.js <URL> <FILENAME>`
	- URL: `valid http url`
	- FILENAME: `CSV file will be saved by this name`
